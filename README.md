# Eason 英语闯关 C 版

iPad / iPhone 触屏离线学习应用

## 在线访问

打开 iPhone / iPad 的 Safari 浏览器，访问本项目的 GitHub Pages 地址即可使用。

## 离线安装

1. Safari 打开网页
2. 点底部分享按钮 ⬆️
3. 选择"添加到主屏幕"
4. 桌面出现 🚀 图标
5. 之后完全离线可用

## 特性

- ✅ 完全离线可用
- ✅ 106 个单词，11 个主题
- ✅ 逐字母拼读 × 10 遍
- ✅ 例句朗读 × 5 遍
- ✅ 触屏优化，全屏运行
- ✅ 进度自动保存
